# Study Planner Dashboard

## Description
A simple web application that helps students plan their study schedule. Users can add subjects, assign study hours, and track their progress.

## Features
- Add and delete subjects
- Calculate total study hours
- Calculate hours per day
- Track progress with a progress bar


## How It Works
- Enter subject name and hours
- View total study hours
- Enter number of days to get daily study plan
- Enter studied hours to see progress
